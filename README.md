# RNN-LSTM-and-Transformer

## Overview

This project is a multi-label emotion classification system built using three different machine learning and deep learning approaches:

- Logistic Regression with TF-IDF
- BiLSTM (PyTorch)
- DistilBERT Transformer

The system predicts multiple emotions from a given text and provides AI-generated emotional guidance using Gemini.

---

## Features

- Multi-label emotion classification
- Detection of 28 emotion categories
- Logistic Regression baseline model
- BiLSTM deep learning model
- DistilBERT Transformer model
- Flask REST API backend
- Interactive web dashboard
- AI-powered emotion coaching
